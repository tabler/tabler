# Avatars

Avatars help customise various elements of a user interface and make the product experience more personalised. They are often used in communication apps, collaboration tools and social media.

## Default markup

Use the `avatar` class to add an avatar to your interface design for greater customisation.

<example centered separated>
<span class="avatar" style={{ backgroundImage: 'url(https://images.unsplash.com/photo-1450297350677-623de575f31c?q=80&fm=jpg&crop=faces&fit=crop&h=100&w=100)'}}></span>
<span class="avatar">JL</span>
<span class="avatar" style={{ backgroundImage: 'url(https://images.unsplash.com/photo-1463453091185-61582044d556?q=80&fm=jpg&crop=faces&fit=crop&h=100&w=100)'}}></span>
</example>

~~~html
<span class="avatar" style="background-image: url(...)"></span>
<span class="avatar">JL</span>
<span class="avatar" style="background-image: url(...)"></span>
~~~
